# Study Japanese 🇯🇵  
(日本語学習アプリ)

---

##  Overview  
Study Japanese is a simple web application designed to help beginners learn basic Japanese vocabulary through flashcards and interactive quizzes.

This project was built as part of my learning process in frontend development.

---

##  Motivation  
I wanted to combine my interest in Japanese language with web development practice.  
The goal was to build something simple, useful, and easy to improve over time.

---

##  Key Features  

- Flashcards for memorizing vocabulary  
- Multiple-choice quiz system  
- Basic score tracking  
- Clean and minimal interface  

---

##  Tech Stack  

- HTML5 – structure  
- CSS3 – styling  
- JavaScript – application logic  

---

##  Development Notes  

This project is still in early development. I plan to expand it gradually by adding more learning features and improving user experience.

---

##  Future Improvements  

- JLPT level system (N5 → N1)  
- Progress saving in browser  
- Vocabulary categories (hiragana, kanji, phrases)  
- UI/UX improvements focused on usability
